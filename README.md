# Bentcam Website

Bentcam - Energy-saving antiperspirant constructions that do not require resistance.


## Features

- Easy to integrate with Backend
- Fully Responsive

## Run Locally

Clone the project

```bash
  git clone https://github.com/keerthi-inala/Bentcam.git
```

Go to the project directory

```bash
  cd Bentcam
  cd client
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```



## Tech Stack

* [React](https://reactjs.org/)
