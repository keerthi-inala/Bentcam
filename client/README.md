# Bentcam Website

Bentcam - Energy-saving antiperspirant constructions that do not require resistance.

Frontend - React
Backend - Java
## Features

- Easy to integrate with Backend
- Fully Responsive

## Run Locally

Clone the project

```bash
  git clone https://github.com/keerthi-inala/Bentcam.git
```

Go to the project directory

```bash
  cd Bentcam
  cd client
```

Install dependencies

```bash
  npm install
```

## Backend API (Java)

The repository now includes a Java backend in `backend/`.

Run backend:

```powershell
cd ..\backend
mvn spring-boot:run
```

Run frontend:

```powershell
npm start
```

With the CRA `proxy` configured, frontend requests to `/api/*` are forwarded to `http://localhost:8080`.

Health check endpoint: `GET http://localhost:3000/api/health` (proxied to backend)

```

Start the server

```bash
  npm start
```
